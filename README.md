# IMkuchou

**アイエム空調株式会社** 見積書・注文書・注文請書・納品書・請求書 統合管理システム

## 概要

- ブラウザ内で完結するサーバーレス構成(データは端末の localStorage に保存)
- 5書類を1つのアプリで管理し、見積書 → 注文書 → 注文請書 → 納品書 → 請求書 と内容を引き継いで作成できるワークフロー
- 各書類はA4レイアウトでプレビューし、PDF書き出しに対応(html2pdf.js)
- 取引先マスタ・品目マスタ・自社情報(振込先/インボイス登録番号など)を管理

## 使い方

`https://uchiyamazion.github.io/IMkuchou/` にアクセスするだけで利用できます。ビルド不要。

1. 「自社設定」で会社情報・振込先・インボイス登録番号を入力
2. 「マスタ管理」で取引先・よく使う品目を登録(任意、都度手入力も可)
3. 各書類タブから「+ 新規〇〇書」を作成し、明細を入力
4. 書類編集画面の「〇〇書を作成 →」ボタンで次の書類へ内容を引き継ぎ
5. 「PDF書き出し」で印刷用PDFをダウンロード

## 注意事項

- データは**この端末のブラウザ内**にのみ保存されます。ブラウザのデータ削除・別端末・シークレットモードでは共有されません。
- 定期的に大事な書類は各自PDF保存しておくことを推奨します。

## 技術構成

- React 18 (UMD) + Babel Standalone(ビルド不要、CDN読込)
- html2pdf.js によるPDF出力
- Google Fonts: Zen Kaku Gothic New / Noto Sans JP / JetBrains Mono
- ビルドプロセスなし、静的ファイルのみで GitHub Pages にそのままデプロイ
